The Inner Gameplay Loop

This is the decision flow happening inside every phase of play. When you need to answer a question, the type of question determines which tool to use. The sidebar tools marked with ✨ are always available.

    ACT --> PF{"Pass / Fail"}
    PF -->|Pass| NEXT["What do you do next?"]
    PF -->|Fail| CONS["Address the Consequence"]
    CONS --> WHAT["What happens next?"]
    NEXT --> ENV
    WHAT --> ENV

Playing Nimble Solo

  1. Create a Main Hero and a Sidekick Hero. Each can have a Companion if desired.
  2. Build a Quest.
  3. Use the Yes / No Oracle and provided tables to answer questions.
  4. Imagine the scene and resolve the situation.

Yes / No Oracle

Use this to answer any Yes / No questions you may have.

  • Roll 2d6
  • Equal Odds: Read Primary (Left) Die
  • Likely Odds: Read Highest Die
  • Unlikely Odds: Read Lowest Die
  • Doubles trigger a Random Event
  • Only 1 Random Event at a time

General Tips for Solo Play

  • Guidelines, not rules. There is no wrong way to play. Choose what is most fun.
  • Keep the flow moving. If something doesn't make sense, reroll, or choose a row.
  • Defer some questions for later; Answers may naturally come to you as you play.
  • Fail-forward: Every failure is a new opportunity and a new challenge to overcome.
  • Be your character. Describe the situation, how your character handles it, and why.
  • Your actions have consequences and should change the world in a meaningful way.
  • Create your own themed 1d10 lists on index cards to customize your adventure.

Luck

+1 Luck on a miss or failed save. Max 5. Earning a 6th resets to 1d4. Reset to 0 on rest.
